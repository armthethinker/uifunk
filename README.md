# uifunk v0.1.0
UIFunk is a content placeholder engine built to fill in content in HTML prototypes.

Currently built in PHP. Will be built into JS.

## Available Content
- **People**: Philosophers
- **Company Names**: Fruit
- **Addresses**: Trees
- **Units**: Units
- **Jobs**: Jobs
- **Cities**: Cities
- **States**: States
- **Projects**: Project names
- **Content**: Random philosophy words
